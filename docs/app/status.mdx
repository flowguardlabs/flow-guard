## Current Network

FlowGuard currently runs on **Chipnet**. The public app, wallet flows, and covenant deployment paths are built around BCH testnet operation today.

| Component | Current state | Network |
| --- | --- | --- |
| Frontend | available | Chipnet |
| Backend API | available | Chipnet |
| WalletConnect v2 | available | Chipnet |
| Contract builds and local verification | available | local + Chipnet-targeted |
| Indexer | optional operator component | Chipnet |
| Executor | optional operator component | Chipnet |

## Current Operational Notes

- the public product is still Chipnet-only
- some operator services are optional and can be self-hosted
- richer analytics and monitoring depend on the indexer and related operator infrastructure
- vault creation still uses the current signer model and product constraints shown in the app
