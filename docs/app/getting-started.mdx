FlowGuard is a BCH-native treasury, streams, distributions, and governance product. The web app connects to your BCH wallet via WalletConnect v2 and lets you deploy and control covenant transactions directly from the browser without backend custody.

## Supported Wallets

FlowGuard works with any WalletConnect v2 compatible BCH wallet:

- **Paytaca**: iOS and Android, full CashTokens support
- **Selene**: mobile and browser extension, full CashTokens support

## Connecting Your Wallet

<Steps>
  <Step title="Open the App">
    Navigate to the FlowGuard frontend URL. The header shows **Connect Wallet**.
  </Step>
  <Step title="Scan or Click">
    Click **Connect Wallet**. On mobile, the wallet app opens automatically. On desktop, scan the QR
    code with your wallet app.
  </Step>
  <Step title="Approve">
    Approve the connection request in your wallet. Your BCH address now appears in the header.
  </Step>
</Steps>

## Network

FlowGuard currently runs on **Chipnet** (BCH testnet). You can get test BCH from the [Chipnet faucet](https://tbch.googol.cash/).

## Choose a Workspace

When you enter the app, FlowGuard lets you choose between:

- **Personal Workspace** for individual vaults, streams, payments, and distributions
- **Organization Workspace** for DAO-style treasury, role, policy, and stream operations

The two workspaces share the same underlying product flows. The difference is context, navigation, and treasury defaults.

## Navigating the App

| Section    | What it does                                                                  |
| ---------- | ----------------------------------------------------------------------------- |
| Dashboard  | Overview of your treasury, streams, payments, and linked workflows            |
| Streams    | Create and manage linear, hybrid, recurring, milestone, and tranche schedules |
| Payments   | Set up recurring payroll or subscription-style payouts                        |
| Airdrops   | Run token distribution campaigns                                              |
| Grants     | Milestone-based budget disbursements                                          |
| Bounties   | Fund tasks with conditional release                                           |
| Rewards    | Issue one-time or ongoing rewards                                             |
| Governance | Vote on vault proposals with token locking                                    |
| Explorer   | Browse all FlowGuard activity on-chain                                        |
| Vaults     | Multi-sig treasury containers                                                 |

## First Steps

1. Get Chipnet BCH from the [faucet](https://tbch.googol.cash/)
2. Choose whether you are working in a personal or organization context
3. Create or open a vault if you want treasury-linked operations
4. Open the stream shape gallery or create a recurring payment
5. Review the schedule preview before signing any funding transaction

## Stream Builder Highlights

The shared stream builder currently supports:

- linear schedules
- linear schedules with cliffs
- hybrid upfront unlock plus linear tail
- recurring schedules
- refillable recurring runway
- equal milestone schedules
- custom bounded tranche schedules
- batch creation with row-level charts and CSV import / export
