FlowGuard is a BCH-native treasury and payments protocol. The web app connects to your BCH wallet via WalletConnect v2 and lets you deploy covenants directly from the browser â€” no backend custody, no seed phrases submitted.

## Supported Wallets

FlowGuard works with any WalletConnect v2 compatible BCH wallet:

- **Paytaca**: iOS and Android, full CashTokens support
- **Selene**: mobile and browser extension, full CashTokens support

## Connecting Your Wallet

<Steps>
  <Step title="Open the App">
    Navigate to the FlowGuard frontend URL. The header shows **Connect Wallet**.
  </Step>
  <Step title="Scan or Click">
    Click **Connect Wallet**. On mobile, the wallet app opens automatically. On desktop, scan the QR code with your wallet app.
  </Step>
  <Step title="Approve">
    Approve the connection request in your wallet. Your BCH address now appears in the header.
  </Step>
</Steps>

## Network

FlowGuard currently runs on **Chipnet** (BCH testnet). You can get test BCH from the [Chipnet faucet](https://tbch.googol.cash/).

## Navigating the App

| Section | What it does |
|---------|-------------|
| Dashboard | Overview of all your active streams, payments, and vaults |
| Streams | Create and manage vesting streams |
| Payments | Set up recurring payroll or subscriptions |
| Airdrops | Run token distribution campaigns |
| Grants | Milestone-based budget disbursements |
| Bounties | Fund tasks with conditional release |
| Rewards | Issue one-time or ongoing rewards |
| Governance | Vote on vault proposals with token locking |
| Explorer | Browse all FlowGuard activity on-chain |
| Vaults | Multi-sig treasury containers |

## First Steps

1. Get Chipnet BCH from the [faucet](https://tbch.googol.cash/)
2. Create a Vault with 3 signer addresses  
3. Fund the vault
4. Create a stream, payment, or airdrop from the vault
