FlowGuard's stream shape gallery is the fastest way to choose the right schedule. Each gallery option maps to a real contract-backed schedule family, not just a visual preset.

## Shape Families

| Shape family       | Use it when you want                                     | What it looks like                   |
| ------------------ | -------------------------------------------------------- | ------------------------------------ |
| **Linear**         | continuous vesting from start to finish                  | a smooth rising unlock curve         |
| **Linear + Cliff** | no claims early, then continuous vesting after the cliff | a flat start, then a smooth rise     |
| **Hybrid**         | a meaningful first unlock plus a vesting tail            | one visible jump, then a smooth rise |
| **Recurring**      | fixed cadence payouts for payroll or retainers           | equal-step payout windows            |
| **Step**           | equal milestone unlocks                                  | evenly spaced unlock steps           |
| **Tranche**        | custom bounded unlock checkpoints                        | uneven staged unlocks                |

## How to Choose

### Linear

Best for:

- simple contributor vesting
- employee equity-style schedules
- token releases without milestone checkpoints

### Linear + Cliff

Best for:

- contributor or investor schedules that should not unlock immediately
- treasury releases that should stay locked until a review point

### Hybrid

Best for:

- grants that need an initial unlock for setup costs
- contributor plans with a meaningful first release
- investor schedules that combine a first unlock with a vesting tail

### Recurring

Best for:

- payroll
- subscriptions
- stipends
- retainers

If the schedule should continue over time without redeploying, use a refillable recurring template.

### Step

Best for:

- equal monthly unlocks
- equal quarterly unlocks
- simple milestone schedules without custom tranche weights

### Tranche

Best for:

- custom grant schedules
- staged investor unlocks
- treasury releases that need custom checkpoint weighting

## What the Gallery Does

The shape gallery can:

- preload a template into the create flow
- show the unlock profile before creation
- help you compare continuous, milestone, recurring, hybrid, and tranche schedules
- open the custom tranche designer for bounded staged schedules

## What the Create Page Adds

Once you enter the builder, the app lets you refine the template with:

- start date
- end date or duration
- cliff
- cadence
- hybrid unlock timing
- upfront percentage
- tranche checkpoint timing
- tranche percentages

The chart on the create page reflects the actual schedule after your edits.
